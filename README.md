# MIDI CC, NRPN, and notes dataset

This is MIDI Guide, the MIDI CC, NRPN, and notes dataset maintained by Pencil Research. We want to document the MIDI implementation of every synthesizer.

You might like to [visit the project's website at https://midi.guide](https://midi.guide/) to browse this data more conveniently.

The portions of this dataset that refer to specific devices may be owned by the devices' respective manufacturers. For everything else, see [LICENSE](LICENSE).

## Contributing

For contribution guidance, please see [CONTRIBUTING.md](CONTRIBUTING.md)

We welcome and value contributions from the community. You can open issues here on GitHub, or send us pull requests from your fork. Alternatively, download and alter any file you like, email it to [midi@midi.guide](mailto:midi@midi.guide), and we'll publish your changes for you.

You can edit the CSV files in this dataset with a spreadsheet program (like Excel), or a text editor.

If you want to document the CCs/NRPNs of a new device, download `template.csv` to get started.

[👉 🎹 Get template.csv 👈](https://raw.githubusercontent.com/pencilresearch/midi/main/template.csv)

Or, if you want to document the MIDI notes of a new device, download `template.triggers.csv` to get started.

[👉 🥁 Get template.triggers.csv 👈](https://raw.githubusercontent.com/pencilresearch/midi/main/template.triggers.csv)

Once your CSV is merged into this repository, your device will appear on MIDI Guide's project website with our thanks!
